Synopsis

uname [-snrvma]

Description

The uname utility writes information about the running system to standard output.

By default, uname writes the operating system name.

Options

-a

Behave as though all of the options -snrvm were specified.

-s

Write the operating system name.

-n

Write the nodename.

-r

Write the operating system release.

-v

Write the operating system version.

-m

Write the system hardware type.

Diagnostics

The following error values are returned:

Error value: 0
Diagnostic: Successful completion.
Error value: >0
Diagnostic: An error occurred.

Standards

The uname utility conforms to the Single UNIX Specification, Version 3 (SUSv3).
